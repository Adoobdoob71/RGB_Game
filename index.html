Can you guess the RGB?
The RGB color model is an additive color model in which the red, green, and blue primary colors of light are added together in various ways to reproduce a broad array of colors. The name of the model comes from the initials of the three additive primary colors, red, green, and blue. The values range from zero to 255. For example, the RGB values for the color red are rgb(255, 0, 0). Your goal is to find the RGB values of the presented color with each value being plus or minus 20 from the correct value.
